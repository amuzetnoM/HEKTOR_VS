# Scope — Competitive & System Comparison

This directory contains materials for the project's scope and competitive comparisons.

Files:
- `COMPETITIVE_COMPARISON.html` — interactive comparative dashboard and export used for competitor analysis.

Purpose:
- Capture the competitive positioning of Hektor (Vector Studio) with benchmarks and feature comparisons.
- Provide an easily discoverable entry point for reviewers and stakeholders.

For the v4.0.0 release, the exhaustive system snapshot is in `docs/research/HEKTOR_ANALYSIS.md` and should be referenced from release notes and public-facing materials.
