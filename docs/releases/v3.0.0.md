## ðŸš€ Quick Start

```bash
# Initialize database
hektor init ./mydb --preset gold-standard

# Add documents
hektor add ./mydb --text "Gold prices rising"
hektor add ./mydb --file document.txt

# Search
hektor search ./mydb "gold outlook" -k 20

# Hybrid search with fusion
hektor hybrid:search ./mydb "analysis" --fusion rrf -k 20

# NLP operations
hektor nlp:embed --text "gold prices" --model all-MiniLM-L6-v2
hektor nlp:ner --text "Gold prices in London reached $2000"

# Interactive mode
hektor shell ./mydb
```
