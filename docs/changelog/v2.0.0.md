# Foundation Release

## Overview

This release establishes the **complete FOSS vector database foundation** with production-ready core infrastructure.

---

## Core Features

### Index Implementations
- **HNSW Index**: High-performance approximate nearest neighbor search
- **FlatIndex**: Exact search for smaller datasets
- **Product Quantization**: Memory-efficient vector compression

### Universal Data Ingestion
- PDF, Text, Excel, XML, Parquet adapters
- SQLite and pgvector integration
- Apache Arrow support for columnar data

### Storage & Persistence
- SQLite-backed metadata storage
- Binary serialization for indices
- Full read/write capabilities

### API & Bindings
- HTTP client adapter
- Python bindings (pyvdb)
- Comprehensive logging system

---

## Build System

- C++23 with CMake
- Cross-platform: Windows (MinGW/MSVC), Linux, macOS
- vcpkg dependency management
